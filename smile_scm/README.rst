This module offers the possibility to create a repository and clone its branches.

Features:

* Creation and management of repositories, branches, Odoo versions, version control systems and tags.
* Possibility to clone and pull branches.

Usage
=====

To create a repository :

#. Go to ``Source Code > Repositories`` menu.

Bug Tracker

Bugs are tracked on `GitHub Issues <https://github.com/Smile-SA/odoo_addons/issues>`_.
